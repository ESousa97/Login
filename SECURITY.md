# Política de Segurança

## Relato de vulnerabilidades

Se você identificar uma vulnerabilidade, por favor reporte de forma responsável.

- Abra uma issue marcada como **security** (sem detalhes sensíveis), ou
- Envie um e-mail para o responsável do repositório.

## Suporte

Este projeto segue o fluxo de correções pelo GitHub Actions e depende de atualizações mensais do Dependabot.
