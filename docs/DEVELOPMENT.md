# Desenvolvimento

## Pré-requisitos

- Node.js 18+ (apenas para lint, testes e build local)

## Comandos

- `npm run lint`: valida HTML/CSS
- `npm run format`: formata arquivos compatíveis
- `npm run test`: executa testes de smoke
- `npm run test:coverage`: gera cobertura
- `npm run build`: cria pasta dist/ com os artefatos

## Estrutura de pastas

- **docs/**: documentação técnica
- **tests/**: testes automatizados
- **scripts/**: utilitários de build e validação
