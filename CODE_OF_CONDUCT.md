# Código de Conduta

Este projeto adota o Contributor Covenant v2.1.

## Nosso compromisso

Prometemos tornar a participação uma experiência livre de assédio para todos, independentemente de idade, tamanho do corpo, deficiência, etnia, identidade e expressão de gênero, nível de experiência, nacionalidade, aparência pessoal, raça, religião ou orientação sexual.

## Padrões esperados

- Use linguagem acolhedora e inclusiva.
- Respeite pontos de vista diferentes.
- Aceite feedback construtivo.

## Aplicação

Para reportar problemas de conduta, abra uma issue confidencial ou envie e-mail para o responsável do projeto.

Texto completo: https://www.contributor-covenant.org/version/2/1/code_of_conduct.html
